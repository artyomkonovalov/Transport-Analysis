# Техническое задание
### Кейс: Анализ городской транспортной сети как графа

**Цель:** 
	применить базовые структуры данных и алгоритмы к реальной задаче анализа транспортной сети города

**Необходимо**:
+ собрать (спарсить) реальные открытые данные о городском транспорте;
+ спроектировать и реализовать собственные структуры данных (граф, словари и т.д.) для хранения маршрутов и остановок;
+ выполнить анализ (поиск путей, пересадок, «узких мест» и др.);
+ подготовить визуализации с использованием Yandex DataLens;
+ оформить результаты в виде работающего проекта, отчёта

**Каждая команда должна ответить на вопросы из списка:**
1.    Топ-остановки по числу маршрутов. Какие 5–10 остановок проходят наибольшее количество маршрутов?
2.    Длины маршрутов. Какой маршрут самый длинный по числу остановок? Самый короткий?
3.    Связность сети. Сколько компонент связности в графе?

**Требования к визуализации:**
+ Карта города с остановками
+ Точки: остановки
+ Цвет/размер точки: число маршрутов через остановку
